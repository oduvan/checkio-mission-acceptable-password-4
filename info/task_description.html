In this mission you need to create a password verification function.

Those are the verification conditions:

  • Cond 1. the length should be bigger than 6;
  • Cond 2. should contain at least one digit, but it cannot consist of just digits;
  • Cond 3. if the password is longer than 9, we don't check [Cond 2].

Input: A string.

Output: A bool.
